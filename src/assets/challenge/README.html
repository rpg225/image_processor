Image Blurring in Rust

Task Overview:

Objective: Develop a Rust program to blur an input image using Gaussian blur.

Input: Accept an image file as input for blurring.

Output: Produce a blurred version of the input image and save it as a new file.

Functionality: Implement Gaussian blur using the image crate.

Parameterization: Allow users to specify the blur strength (kernel size).

Error Handling: Ensure robust error handling for file operations.

Documentation: Include clear comments and instructions for usage.

Guidelines:

Use of Libraries: Utilize the image crate for image processing.

Modularity: Design modular function

Testing: Thoroughly test the program with various images and blur strengths.

Code Quality: Follow Rust conventions for clean and efficient code.

User Interaction: Provide clear instructions for input file, output file, and blur strength.
